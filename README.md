MERN Product Management System

A full-stack Product Management System built using the **MERN Stack (MongoDB, Express, React, Node.js)**.  
This application allows users to manage products efficiently with features like adding, editing, deleting, filtering, and sorting products.

---

## Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- DaisyUI
- Axios
- React Router
- React Hot Toast
- Lucide React Icons

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- REST API

---

# Features

- Add new products
- View all products in a card layout
- Edit product details
- Delete products with confirmation modal
- Filter products by category
- Sort products by price
- Dynamic category dropdown
- Toast notifications for actions
- Responsive UI design
- Modern card hover animations
